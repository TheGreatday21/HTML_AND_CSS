For this website we are going to learn the different types of css

Inline css
This css is defined by using style function in a tag .Hence the name inline


Internal css

This involves using the style tags in the head section of the html file . Mainly used if the page has a unique design



External css
We use another file called a style sheet and link it to the html file in the head section using the link function
